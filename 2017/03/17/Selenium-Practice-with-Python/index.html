等待

刚接触Selenium的人经常会遇到这样的困惑：为什么我手动一行一行执行的时候正常，写到一个文件中执行就报错？原因是，前一行执行完，但页面却没有加载好，导致执行后一行时，相关元素还没有加载出来。起初，大家可能会想到time.sleep()，但这毕竟不是一个好方法。后来查阅文档发现，Selenium已经提供了等待相关的函数。
